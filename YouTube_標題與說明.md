## 📝 影片說明（直接複製貼上）

開完一場會，會議記錄、決議、還有每個人的待辦——自己長出來。

這是我（Morton）獨立開發的「Google會議自動化助理」。把會議錄音丟進 Google Drive，背景就自動產出：
・會議記錄 Google Doc（摘要 / 決策 / 待辦）
・每個人的 Google Tasks 待辦（含截止日）
・一封完成通知 Email
全程零人工，不用按任何按鈕。

▍怎麼運作
1. 錄音丟進指定的 Drive 資料夾
2. Google Apps Script 每 5 分鐘自動偵測
3. Gemini（Multi-Agent）分析：摘要 / 決議 / 待辦
4. 自動產出 Doc ＋ Tasks ＋ Email

▍技術
Google Apps Script（事件驅動排程）＋ Gemini（多代理、結構化 JSON）＋ Drive / Docs / Tasks API ＋ Gmail。零伺服器，用現有的 Google Workspace 就能跑。
※ 誠實標示：目前端到端運行的是 Google 生態系路徑；Microsoft 365 版本架構已設計、整合中。

它的起點是我先前的轉錄小工具 VidoSTT——我發現轉錄只是第一步，真正花時間的是「會後整理」，於是把它升級成這條全自動工作流。
